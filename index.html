To add the signature, you've copied as your default signature in Outlook, follow these steps:

  1. Open Outlook and select 'New Email'
  2. In the ribbon menu, click 'Insert' then 'Signature > Signatures...'
  3. Choose 'New' and type a name for the signature
  4. Paste the copied signature into the box using Ctrl+V. Important: Make sure you keep source formatting when pasting
  5. Set this new signature as your default by choosing it for new messages and replies in the Set Default Signature options
  6. Click 'Save' to save changes.
